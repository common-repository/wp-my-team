== Description ==

The main purpose of the plugin is to display team/staff members. You can display a set of pictures and information in different layouts. Although it was built thinking of a team/people structure, it can be used to display other kinds of information that require an image and additional information.


== Installation ==

1. Upload my-team folder to wp-content/plugins.
2. Click "Activate" in the WordPress plugins menu.
3. New menu tab "Team" is created in WordPress admin
